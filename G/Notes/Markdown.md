…………………………以此类推

# 四.列表

## 1.有序列表

```
1. （后加空格）
Ctrl+shift+[
```

1. 一
2. 二

## 2.无序列表

```
+ 或- 或* （后有空格）
Ctrl+shift+]
```

* 1
* 2

## 3.任务列表

```
- [ ] (注意空格)
```

- [ ] 1
- [ ] 2
