世界上最大的蛇确实是森蚺，但最长的蛇却不是森蚺！森蚺和网纹蟒谁厉害？
世界上最大的蛇确实是森蚺，但最长的蛇却不是森蚺，而是网纹蟒！尽管它们都是蛇类，但两者所专长的技能也不一样，当然关键还是它们生活习性不尽相同！亚马逊森蚺，与各位见到的一样，亚马逊森蚺喜欢生活在沼泽地区，这是它们最喜欢的环境！
网纹蟒是地球上最长的蛇类，它的一生都在生长！最长的网纹蟒能长到14M，网纹蟒的绞杀能力是蛇类中最强的！网纹虽然在陆地上速度更快,但森蚺体格更粗壮,绞杀力量更强.如果在陆地上,森蚺可能会在被网纹狠咬几口之后把网纹干掉.如果是在水里,呵呵网纹就死得惨了。
在同等长度什么蟒都不是森蚺的对手。
森蚺和网纹蟒不生活在同一环境中，那么说明他们不是竞争对手所以就是各有千秋了。它们是为适应不同环境而生，两者间没有可比性。森蚺和网纹蟒是具有鲜明特点的两类体形巨大的蛇类，森蚺体重最大、网纹蟒体型最长。森蚺主要生活在南美洲亚马孙大森林，而网纹蟒主要生活在东南亚森林地带，也有少部分分布在亚马孙河流域，因此，二者在地理空间的分布上有重叠的可能，如果二者相遇，谁是最利害的“矛”呢？
森蚺隶属于蛇亚目蚺科水蚺属，身体躯干比较肥壮，皮肤颜色呈橄榄色或者绿色，分布有黑色的圆形斑纹，因此也被称为绿森蚺。其身长普遍在4米以上，有的甚至能达到8－9米，平均体重在80公斤左右，最重的可能达到300公斤。它没有毒性，但长期在亚马孙热带森林中生活，进化出了拥有巨大缠绕力、非常善于游泳、吞咽能力惊人等本领，经常以乌龟、水豚、鸟类等为食，有时甚至可以吞食鳄鱼。
据研究，森蚺的缠绕力可以达到每平方英寸130多磅，相当于在猎物身上放置将近10吨重的巨石，一旦猎物被它们缠住，瞬间骨骼就会被压碎，呼吸困难，最后血液停止流动、呼吸衰竭而死。森蚺分布于南美洲亚马逊雨林，森蚺的平均体型可以长到5.21米，平均体重约为98公斤，最大重量可达三四百公斤。有人声称森蚺可以长到8到10米，这是非常大的，但这些是一些未经证实的事实。
网纹蟒隶属于蛇亚目蟒科蟒属，身体异常粗长，皮肤上分布有菱形的网格状花纹，头顶部有一条明显的竖线。其身长一般在6米以上，最长的可以达到12米左右，但是体重相对较轻，平均体重40公斤左右，最重的有只有100公斤。它的食物主要是一些小型的动物，也有捕食鹿类等大型动物的时候，不过网纹蟒的天敌包括鳄鱼，在这一点上，其战斗力要逊色于森蚺。
如果森蚺和网纹蟒相遇并争斗的话，虽然网纹蟒在身长方面可能会占据优势，但在体重和缠绕力上，森蚺要更胜一筹。另外，森蚺的吞咽能力也更为强大，游泳技能也更加出众，所以无论是在水中还是陆上，森蚺绝对会力压网纹蟒而取得最终的胜利。
大蛇为蟒，大蟒为蚺！首先就区分二者了！蟒蛇胃口比较大，可以吞食比较大的动物，一口咬住，然后缠绕，导致动物心脏停止跳动而死亡！蚺是把猎物勒死，用躯体把猎物盘绕住，利用强有力发达的肌肉力量勒死猎物！如果两者遇到，在陆地上森蚺可能被网纹蟒咬上两口，但是给快蚺会利用强大的身躯盘绕蟒蛇，最后把它勒死，如果在水里，蚺蛇活动更加迅速，更容易杀死蟒蛇。
根据以上情况，我们可以说森蚺比网纹蟒更有力量，如果互相争斗，可以杀死网纹蟒。
